# Art Gallery Backend (Go)

Это мой первый бэкенд на Go, написанный для интеграции с iOS-приложением.

### Функционал:
- **Галерея**: Просмотр всех картин и фильтрация по автору.
- **Поиск**: Полнотекстовый поиск по названиям картин.
- **Авторы**: Список авторов с их портретами и биографией.
- **Управление**: Добавление, удаление и лайки (Favorite).

### Как запустить:
1. Клонировать репозиторий.
2. Запустить команду `go run main.go`.
3. Сервер будет доступен по адресу `http://localhost:8080`.
